```bash

```

---

## 🎮 How to Play

1. Press any key to start the game.
2. Watch the highlighted color carefully.
3. Click the boxes in the exact same order.
4. Every level adds one more color to the sequence.
5. A wrong click ends the game.
6. Try to beat your highest score!
